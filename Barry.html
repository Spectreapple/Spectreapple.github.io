Barry is a 2D platform puzzle game
It's a short but impacting brain game where
you as the player are being challenged to use
Barry's unique abilities
This project is also made in unity using c#

Barry was made in a timewindow of 5 weeks,
all the assets are selfmade, I learned that time management
is the most important asset in game development
The workflow MosCow is used while making this project and everything
is carefully documented.
